# 📊 NextWave KPI 하락 원인 분석 프로젝트

## 1. 프로젝트 개요

본 프로젝트는 생산성 SaaS 서비스 NextWave의 핵심 KPI 하락 원인을 분석하고,  
데이터 기반 개선 전략을 도출하는 것을 목표로 진행되었습니다.

최근 4개월 동안 다음 KPI가 동시에 하락하는 문제가 발생했습니다:

- 회원가입 전환율 ↓
- 무료체험 시작률 ↓
- 유료 전환율 ↓
- 4주차 유지율 ↓

본 분석은 단순 지표 확인을 넘어,  
퍼널 구조, 사용자 행동, 채널 특성을 종합적으로 분석하여  
실제 비즈니스 문제를 해결하는 데 초점을 맞췄습니다.

---

## 2. 핵심 문제 정의

> NextWave의 KPI 하락은 유입 감소가 아닌  
> 가입 퍼널 이탈 + 초기 가치 전달 실패 + 고객-제품 미스매치에서 발생

---

## 3. 데이터셋 구성

| 데이터셋 | 설명 |
|----------|------|
| funnel_summary | 월별 전체 퍼널 요약 |
| channel_performance | 유입 채널별 성과 |
| landing_event_log | 랜딩~가입 단계 이벤트 |
| trial_usage_summary | 기능별 사용 및 전환 |
| segment_retention | 고객 유형별 유지율 |
| customer_feedback | 사용자 피드백 |

---

## 4. 분석 과정

### 4.1 EDA (탐색적 분석)

- KPI 전반적 하락 확인
- 방문자 수는 증가 (+7.7%)
- 퍼널 초입에서 대규모 이탈 발생

---

### 4.2 퍼널 분석

- CTA 클릭 이전 이탈: 74%
- 이메일 인증 단계 이탈: 51%

👉 대부분의 유저가 가입 절차 이전 단계에서 이탈

---

### 4.3 가설 수립 및 검증

#### 가설 1. 랜딩페이지 설득 실패
- CTA 클릭률 26%
- 가입 동기 부족 확인

#### 가설 2. 핵심 기능 미노출
- 전환율 높은 기능(알림자동화 41%) 사용률 낮음
- 온보딩 구조 문제

#### 가설 3. SNS 채널 문제
- SNS 유료 전환율 0.7% (최저)
- 지인추천 2.8%

👉 단순 채널 성과 문제가 아닌  
제품과 맞지 않는 고객 유입 문제

---

## 5. 핵심 인사이트

### 1. 퍼널 최상단에서 가치 전달 실패
→ 사용자 대부분이 가입 이유를 느끼지 못함

### 2. 고전환 기능이 노출되지 않는 구조
→ 체험 단계에서 전환 기회를 놓침

### 3. 고객-제품 미스매치
→ 팀 협업 SaaS임에도 개인 사용자 중심 유입

---

## 6. 개선 전략

### 6.1 랜딩페이지 개선 (우선순위 1)
- 가치 제안 메시지 A/B 테스트
- 협업 기능 데모 삽입

### 6.2 채널 전략 개선
- SNS 타겟을 팀 사용자 중심으로 재설계
- 지인추천 프로그램 강화

### 6.3 온보딩 개선
- 팀 초대 기능 우선 노출
- 협업 기능 중심 사용자 흐름 설계
